# 🎯 Interview Copilot

A structured AI-powered interview preparation tool designed to simulate real interview experiences.

## 🚀 Features

- 🎤 Mock Interview Mode
- 🧠 AI-powered feedback on answers
- ⚡ Quick action prompts (DSA, HR, Tips)
- 🎤 Voice input support
- 🎨 Modern responsive UI

## 🧠 Why I Built This

Instead of creating a generic chatbot, I designed a guided interview experience where users interact through structured flows like mock interviews. This makes the product feel purposeful and closer to real-world usage.

## 🛠 Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- OpenRouter API (GPT-based)
